## 二、推荐学习路径

### 第一阶段：数学基础准备 (1-2周)

**需要掌握的前置知识：**
- 概率论基础：高斯分布、条件概率、贝叶斯定理
- KL 散度的定义与性质
- 期望的计算与蒙特卡洛估计
- 基本的神经网络知识

**学习重点：**
- 仔细阅读第1章 VAE 部分
- 手推 ELBO 的完整推导 (Theorem 1.1, 1.2)
- 理解重参数化技巧为什么能让梯度可计算

### 第二阶段：DDPM 核心理论 (2-3周)

**学习重点：**
- 前向过程：理解如何逐步加噪声
  - 关键公式：q(x_t|x_0) = N(√α_t x_0, (1-α_t)I)
- 逆向过程：理解如何逐步去噪
- ELBO 在 DDPM 中的应用
- 为什么预测噪声 ε 等价于预测分数 ∇log p(x)

**建议：**
- 手推 Theorem 2.1 - 2.5
- 画出前向/逆向过程的示意图帮助理解

### 第三阶段：代码实践 (2-3周)

**实践项目：**
1. 用 PyTorch 实现简单的 DDPM
2. 在 MNIST 数据集上训练
3. 可视化前向加噪和逆向去噪过程
4. 生成手写数字图像

**代码框架建议：**
```python
# 核心组件
1. 噪声调度器 (Noise Scheduler)
2. U-Net 噪声预测网络
3. 训练循环
4. 采样循环
```

### 第四阶段：进阶内容 (2-3周)

**DDIM 加速采样：**
- 理解 DDIM 与 DDPM 的区别
- 实现 DDIM 采样器
- 对比不同采样步数的效果

**分数匹配：**
- 学习第3章 SMLD
- 理解分数函数的物理意义
- 了解 denoising score matching

**SDE 统一框架：**
- 理解 SDE 如何统一 DDPM 和 SMLD
- 了解 VP-SDE 和 VE-SDE

## 三、关键公式速查

### VAE
```
ELBO(x) = E_q[log p(x|z)] - D_KL(q(z|x) || p(z))
```

### DDPM 前向过程
```
q(x_t|x_0) = N(x_t; √ᾱ_t x_0, (1-ᾱ_t)I)
x_t = √ᾱ_t x_0 + √(1-ᾱ_t) ε,  ε ~ N(0,I)
```

### DDPM 逆向过程
```
p_θ(x_{t-1}|x_t) = N(x_{t-1}; μ_θ(x_t, t), σ_t²I)
```

### 训练目标
```
L = E_{t,x_0,ε}[||ε - ε_θ(x_t, t)||²]
```

### 分数函数
```
s(x) = ∇_x log p(x)
```

## 五、学习建议

1. **不要跳过数学推导**：Diffusion 模型的美在于其数学优雅性，手推公式能加深理解

2. **边学边实践**：每学完一个概念就尝试用代码实现

3. **画图辅助理解**：前向/逆向过程、噪声调度等都适合用图来理解

4. **从简单开始**：先在 MNIST 上跑通，再尝试更复杂的数据集

5. **关注直觉**：理解"为什么这样做"比"怎么做"更重要
